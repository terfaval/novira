NOVIRA -- Author Palette Registry v2.0 (Public Domain -- Prose Edition)

Scope: This registry includes only public domain prose authors.
Lyrical/poetry-focused palettes are excluded.

System Logic: Authors are grouped into tonal families to preserve brand
coherence. Each author uses 2--3 tones from their assigned palette.
Maximum 3 tones per cover. Muted saturation only (40--60%). No pure
black. No pure white.
